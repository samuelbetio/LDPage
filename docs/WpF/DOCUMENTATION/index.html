C) CSS Files and Structure - top

Styles

We're using 7 CSS files in this template. we have separated styles for improving convenience to edit this template. If you're going to use this template in production we recommend you merge all stylesheets in one file (for decreasing http queries).

Lets take a look closely at our stylesheet files.

bootstrap.min.css

This Stylesheet contains responsive grid system ,default element style based on bootstrap.It is bootstrap framework main css file

font-awesome.min.css

Base styles for FontAwesome icon fonts.

default-theme.css

This is default theme color file for this template.

slick.css

Styles for full screen & carousel slider (slick slider).

photoswipe.css

Photoswipe image gallery main css file.

default-skin.css

Photoswipe image gallery default skin css file.

style.css

Common css style and structures for this template.
